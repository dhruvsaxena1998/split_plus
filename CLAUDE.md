# Split+

Expense splitting app with Go backend and React frontend.

## Quick Reference

| Task           | Command                                         |
| -------------- | ----------------------------------------------- |
| Run tests      | `just test`                                     |
| Build          | `just go-build` / `just fe-build`               |
| Format & lint  | `just be-fmt && just be-lint` / `just fe-check` |
| Start services | `just up`                                       |
